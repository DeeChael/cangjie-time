# time

## 使用教程

### 使用单位后缀创建 Duration
在数字后添加单位即可转换为 Duration
```cangjie
let duration = 100.seconds // 创建一个 100 秒的 Duration
```

| 后缀 | 单位 |
|-----|------|
| days | 天 |
| hours | 小时 |
| minutes | 分钟 |
| seconds | 秒 |
| milliseconds | 毫秒 |
| ms | 毫秒 |
| microseconds | 微秒 |
| μs | 微秒 |
| nanoseconds | 纳秒 |
| ns | 纳秒 |
